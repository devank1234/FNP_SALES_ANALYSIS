# 🌸 Ferns & Petals Sales Analysis Dashboard

## 📌 Project Overview

This project focuses on analyzing sales data from Ferns & Petals (FNP), a gifting company that delivers gifts for various occasions such as Diwali, Raksha Bandhan, Holi, Valentine's Day, Birthdays, and Anniversaries.

The objective of this project was to clean and transform raw data, perform business analysis, and build an interactive Excel dashboard that provides actionable insights into sales performance, customer behavior, product performance, and delivery efficiency.

---

## 🎯 Business Problem

Ferns & Petals wanted to understand:

- Overall revenue performance
- Customer spending patterns
- Product performance across categories
- Delivery efficiency
- Seasonal and occasion-based sales trends
- City-wise order distribution

The goal was to convert raw business data into meaningful insights that can support data-driven decision making.
